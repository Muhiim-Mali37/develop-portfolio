OVERVIEW

For this assignment, our task is to develop an interactive interface, use interface components, and tie the components to an internal data state. I chose to create a list of items for purchase and a cart that totals their price. This website is designed for disney fans. They can shop for mickey mouse sweaters, pants and shirts. They're able to sort and filter the items based on some of the fields of the item

Features:

  • There are 15 items, each displaying item image, name and a description about the item.

  • They can also sort the items from highest to lowest price and vice versa.
  • There is an aggregator (cart) on the same page, displaying items I added, their price and total cost.
